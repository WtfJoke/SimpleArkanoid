Supported Features:
- Grundlegendes Gameplay funktioniert (Verhalten des Balls ähnlich zum Original, Korrekte Erkennung wenn Steine oder Paddle getroffen werden, Erkennung falls der Ball verfehlt wird)
- Verschiedenfarbige Steine, Steine die mehrfach getroffen werden müssen.
- Powerups (Paddle größer, mehrere Bälle, Paddle kleiner)
- Leben
- Punktezähler
